Práctica: Salidas de Error con PHPUnit
Descripción

Este proyecto implementa pruebas unitarias utilizando PHPUnit para verificar que una aplicación genere correctamente errores y excepciones cuando recibe datos inválidos o se presentan situaciones inesperadas.

El objetivo es garantizar que el sistema responda de forma segura y controlada ante condiciones de error.

Objetivos
Validar el lanzamiento de excepciones.
Comprobar los mensajes de error esperados.
Verificar que los métodos detecten entradas inválidas.
Asegurar el correcto funcionamiento del manejo de errores.
Requisitos
PHP 8.x o superior.
Composer.
PHPUnit instalado mediante Composer.
